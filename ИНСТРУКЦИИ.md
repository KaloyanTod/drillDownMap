# Инструкции за стартиране на проекта

## Изисквания

- **Node.js** (версия 18 или по-нова) 
- **npm** 

## Стъпки за стартиране
```
npm install
npm run dev
```

## Регенериране на ADM0 данните

`public/Resources/ADM0/world_adm0_simplified.geojson` е опростен (mapshaper) вариант на суровите ADM0 граници — държи се малък нарочно, за да зарежда бързо картата на ниво "world". Оригиналният (несимплифициран) файл не се пази в repo-то, а само в git историята, преди опростяването.

За да регенерирате файла от нов суров източник:
```
npm run simplify:adm0 --source=path/to/raw_world_adm0.geojson
```
Скриптът пуска `mapshaper` с topology-aware опростяване (`-simplify 4% keep-shapes`) и презаписва `public/Resources/ADM0/world_adm0_simplified.geojson`. След регенериране, проверете визуално картата (особено малки/островни държави) и че броят на features остава същия.
