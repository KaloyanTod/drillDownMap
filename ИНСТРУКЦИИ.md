# Инструкции за стартиране на проекта

## Изисквания

- **Node.js** (версия 18 или по-нова) 
- **npm** 

## Стъпки за стартиране
```
npm install
npm run dev
```
## Структура на проекта

```
for_export/
├── public/
│   └── Resources/
│       ├── ADM0/          # GeoJSON данни за световни граници (ниво 0)
│       └── ADM1/          # GeoJSON данни по държави (ниво 1 — региони)
├── src/
│   ├── assets/            # Компоненти за картата и филтрите
│   ├── config/            # Конфигурация на картата
│   ├── context/           # React контекст и управление на състоянието
│   ├── App.jsx            # Главен компонент
│   ├── App.css
│   ├── main.jsx           # Входна точка
│   └── index.css
├── index.html
├── package.json           # Зависимости и скриптове
├── vite.config.js
├── tailwind.config.js
└── ИНСТРУКЦИИ.md          # Този файл
``
